# ReactJS form create using Formik


## To install Formik

```javascript
      npm i formik      
```

## To install Yup


```javascript
      npm i yup
```

It is used for validation form inputs


## To install nanoid


```javascript
      npm i nanoid  
```

It is used to generate unique id/key.

We used this for key existed in element inside array.map()
